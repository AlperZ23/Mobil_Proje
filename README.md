# Mobil Proje

Flutter ile hazirlanmis mobil uygulama proje iskeleti.

## Kullanilan Teknolojiler

- Flutter SDK
- Dart SDK
- Material Design
- Harici paket kullanilmamistir.

## Kullanilan Flutter Surumu

```bash
Flutter 3.x.x
Dart SDK: Flutter SDK ile gelen surum
```

Bu ortamda dogrulanan surum:

```bash
Flutter 3.44.2
Dart 3.12.2
```

## Proje Klasor Yapisi

```bash
lib/
|-- main.dart
|-- models/
|   `-- product_model.dart
|-- data/
|   `-- mock_data.dart
`-- screens/
    |-- home_screen.dart
    `-- detail_screen.dart
```

## Calistirma

```bash
flutter pub get
flutter run
```
